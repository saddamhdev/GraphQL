# 🧠 1. Main Concept

👉 GraphQL query **frontend / client side এ লেখা হয়**
👉 Backend শুধু schema + resolver দেয়

---

# 🌐 3. Option 2: React / Frontend

👉 React এ Apollo Client use করলে:

```javascript
import { gql, useQuery } from "@apollo/client";

const GET_USER = gql`
  query {
    getUser(id: 1) {
      id
      name
      email
    }
  }
`;

function User() {
  const { data, loading } = useQuery(GET_USER);

  if (loading) return <p>Loading...</p>;

  return <div>{data.getUser.name}</div>;
}
```

✔ Real app এ এইভাবে use হয়

# ⚙️ 5. Backend (Spring Boot)

👉 এখানে query লেখা হয় না ❌
👉 শুধু schema + resolver থাকে

```java
@QueryMapping
public User getUser(Long id) {
    return userService.findById(id);
}
```

# 🔥 6. Real Flow

```text
Frontend (query লিখে)
      ↓
POST /graphql
      ↓
Spring Boot GraphQL
      ↓
Resolver call
      ↓
Response JSON
```
