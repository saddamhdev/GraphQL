## 5Ô∏è‚É£ Resolver (GraphQL Controller)

üìÅ `controller/UserGraphQLController.java`

```java
        
import com.example.election.model.User;
import org.springframework.graphql.data.method.annotation.Argument;
import org.springframework.graphql.data.method.annotation.QueryMapping;
import org.springframework.graphql.data.method.annotation.MutationMapping;
import org.springframework.stereotype.Controller;

import java.util.ArrayList;
import java.util.List;

@Controller
public class UserGraphQLController {

    private List<User> users = new ArrayList<>();

    public UserGraphQLController() {
        users.add(new User(1L, "Admin", "ADMIN"));
        users.add(new User(2L, "Officer", "OFFICER"));

    }

    @QueryMapping
    public List<User> users() {
        return users;
    }

    @QueryMapping
    public User user(@Argument Long id) {
        return users.stream()
                .filter(u -> u.id().equals(id))
                .findFirst()
                .orElse(null);
    }

    @MutationMapping
    public User createUser(
            @Argument String name,
            @Argument String role
    ) {
        User user = new User(
                (long) (users.size() + 1),
                name,
                role
        );
        users.add(user);
        return user;
    }

}


```

üìå ‡¶è‡¶ñ‡¶æ‡¶®‡ßá:

* `@QueryMapping` = GET-like
* `@MutationMapping` = POST-like
* No URL, no `/api/...`
